Badges

Documentation and examples for badges, our small count and labeling component. Badges scale to match the size of the immediate parent element by using relative font sizing and em units.

Examples

Add any of the below mentioned modifier classes to change the appearance of a badge. It can be useful if we would like to display categories or user ranks etc.

Primary Secondary Success Danger Warning Info Light Dark Purple 
<span class="badge badge-primary">Primary</span>
