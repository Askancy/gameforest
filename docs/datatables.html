DataTables

DataTables is a plug-in for the jQuery Javascript library. It is a highly flexible tool, build upon the foundations of progressive enhancement, that adds all of these advanced features to any HTML table.

Introduction

To getting started we need to add the required stylesheet and javascript files to our page. You can find an example below but for more examples check the datatables website.

<!-- Stylesheet -->
<link rel="stylesheet" href="plugins/datatables/css/dataTables.bootstrap4.min.css">

<!-- Javascript -->
<script src="plugins/datatables/js/jquery.dataTables.min.js"></script>
<script src="plugins/datatables/js/dataTables.bootstrap4.min.js"></script>
<script>
    $(function() {
        $("#table_id").DataTable();
    });
</script>
