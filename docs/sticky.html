Sticky

Make elements remain at the top of the viewport, like a sticky navigation. For more examples please check the official plugin documentation website.

Example

To create an element that remains at the top of the viewport when scrolling down the site, add the ya-sticky attribute to that element.

A simple primary alert—check it out!
<div class="alert alert-light" ya-sticky="top: 100" role="alert">A simple primary alert—check it out!</div>

Options

Options can be passed via data attributes. For data attributes, append the option name to ya-ya-sticky. Default option names and values are available below in the table.

option value default description
top integer 60 Pixels between the page top and the element's top.
bottom integer null Pixels between the page bottom and the element's bottom.
zindex integer 9 controls z-index of the sticked element.
