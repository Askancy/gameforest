DatePicker

Flatpickr is a lightweight and powerful datetime picker. Lean, UX-driven, and extensible, yet it doesnâ€™t depend on any libraries.

Introduction

To getting started we need to add the required stylesheet and javascript files to our page. You can find an example below but for more examples check the flatpickr website.

<!-- Stylesheet -->
<link rel="stylesheet" href="plugins/flatpickr/flatpickr.min.css">

<!-- Javascript -->
<script src="plugins/flatpickr/flatpickr.min.js"></script>
<script>
    $(function() {
        $(".datepicker").flatpickr({
            allowInput:true
        });
    });
</script>

Example

We added .datepicker class to the input but you can rename the datepicker class anytime.

<input type="text" class="form-control datepicker">

DateTime

It is also possible to enable time for datepicker plugin by adding true value for enableTime option just like in the example below.

<input type="text" class="form-control datetimepicker">

<!-- Javascript -->
<script>
    $(".datetimepicker").flatpickr({
        allowInput: true,
        enableTime: true,
        dateFormat: "Y-m-d H:i"
    });
</script>

Inline calendar

Display the calendar in an always-open state with the inline option.

<input type="text" class="form-control inlinepicker">

<!-- Javascript -->
<script>
    $(".inlinepicker").flatpickr({
        allowInput: true,
        inline: true
    });
</script>

Time Picker

Use the noCalendar option with true value for hide selectable dates and add only time to the picker.

<input type="text" class="form-control timepicker">

<!-- Javascript -->
<script>
    $(".timepicker").flatpickr({
        allowInput: true,
        enableTime: true,
        noCalendar: true,
        dateFormat: "H:i"
    });
</script>

Custom selector

Add wrap option to the picker with required data attributes to get the same results just like in the example.

<div class="input-group custompicker">
    <div class="input-group-prepend">
        <button type="button" class="btn btn-light btn-icon border-right-0" data-toggle><i class="ya ya-calendar"></i></button>
    </div>
    <input type="text" class="form-control" id="exampleSelect3" data-input>
</div>

<!-- Javascript -->
<script>
    $(".custompicker").flatpickr({
        allowInput: true,
        wrap: true
    });
</script>
