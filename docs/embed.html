Embed

Allow browsers to determine video or slideshow dimensions based on the width of their containing block by creating an intrinsic ratio that will properly scale on any device.

Embed video

Add video to your section or blog post with ya-embed data attribute. Set the custom video thumbnail with the img attribute. Copy youtube url from browser bar and paste it into the data attribute and check the example below.

<div ya-embed="https://www.youtube.com/watch?v=vjF9GgrY9c0">
    <img src="https://img.youtube.com/vi/vjF9GgrY9c0/maxresdefault.jpg" alt="">
</div>

Caption

Add title or length to the embedded video by using ya-title or ya-length attribute. It is also possible to customize the looks of caption with scss just check scss part.

<div ya-embed="https://www.youtube.com/watch?v=dGl00hItvbw" ya-title="Forza Horizon 4 E3 2018: Announce Trailer" ya-length="15:05">
    <img src="https://img.youtube.com/vi/MVTHYMkDvIs/maxresdefault.jpg" alt="">
</div>

Controls

Controls are not hidden by default but we can use the ya-options data attribute to set the correct value for that action. So an example use ya-options="controls: 0" after the ya-embed attribute. For more options check the options part below.

<div ya-embed="https://www.youtube.com/watch?v=Dw_oH5oiUSE" ya-option="controls: 0">
    <img src="https://img.youtube.com/vi/-DBi5FC1hXs/maxresdefault.jpg" alt="">
</div>

Embed twitch

We can use another platform like twitch to embed videos or clips. Don't forget options are not available for embedded twitch players.

<div ya-embed="https://clips.twitch.tv/EncouragingShakingPoxRalpherZ">
    <img src="https://img.youtube.com/vi/JJztiuDIKBo/maxresdefault.jpg" alt="">
</div>

Options

Options can be passed via data attributes. For data attributes, append the option name to ya-option. Default option names and values are available below in the table.

ya-option="controls: 0"
option value default description
autoplay integer 1 Play the video automatically when user clicked to the thumbnail.
controls integer 1 Display the player pause, volume control or timeline.
recommends integer 0 Display recommended playlist end of the video.
mute integer 0 Mute the embedded video by default.
loop number 0 Restart the video if it has ended.
start number null Start to play the video from this time.
