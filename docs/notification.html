Notification

Documentation for built in notificiation with bootstrap alert component. Easy to use just check the available examples below.

Example

Add a notification alert by using ya-notify and ya-title attribute together on an element like in the example below.

<button class="btn btn-outline" ya-notify ya-title="A simple primary alert—check it out!">Notify me</button>

Alignment

Set the notification position by adding align: * option to the ya-notify attribute like in the example below and it will be display in the right place.

<button class="btn btn-outline" ya-notify="align: left" ya-title="A simple primary alert—check it out!">Top left</button>
<button class="btn btn-outline" ya-notify="align: right" ya-title="A simple primary alert—check it out!">Top right</button>
<button class="btn btn-outline" ya-notify ya-title="A simple primary alert—check it out!">Top center</button>
<button class="btn btn-outline" ya-notify="align: bottom-left" ya-title="A simple primary alert—check it out!">Bottom left</button>
<button class="btn btn-outline" ya-notify="align: bottom-right" ya-title="A simple primary alert—check it out!">Bottom right</button>
<button class="btn btn-outline" ya-notify="align: bottom" ya-title="A simple primary alert—check it out!">Bottom center</button>

Variations

Set the notification alert type by adding alert * option to the ya-notify attribute like in the example below.

<button class="btn btn-primary" ya-notify ya-title="A simple primary alert—check it out!">Primary</button>
<button class="btn btn-secondary" ya-notify="alert: secondary" ya-title="A simple secondary alert—check it out!">Secondary</button>
<button class="btn btn-success" ya-notify="alert: success" ya-title="A simple success alert—check it out!">Success</button>
<button class="btn btn-danger" ya-notify="alert: danger" ya-title="A simple danger alert—check it out!">Danger</button>
<button class="btn btn-warning" ya-notify="alert: warning" ya-title="A simple warning alert—check it out!">Warning</button>
<button class="btn btn-info" ya-notify="alert: info" ya-title="A simple info alert—check it out!">Info</button>
<button class="btn btn-dark" ya-notify="alert: dark" ya-title="A simple dark alert—check it out!">Dark</button>

Options

Options can be passed via data attributes. For data attributes, append the option name to ya-notify. Default option names and values are available below in the table.

option value default description
align string center Display the alert component on selected place.
alert string primary Display the alert component on different variation.
