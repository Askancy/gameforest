Buttons

Buttons are used for actions, like in forms, while textual hyperlinks are used for destinations, or moving from one page to another.

Examples

Included several predefined button styles, each serving its own semantic purpose, with a few extras thrown in for more control. Use any of the available button classes to quickly create a styled button.

<button type="button" class="btn btn-primary">Primary</button>
<button type="button" class="btn btn-secondary">Secondary</button>
<button type="button" class="btn btn-success">Success</button>
<button type="button" class="btn btn-danger">Danger</button>
<button type="button" class="btn btn-warning">Warning</button>
<button type="button" class="btn btn-info">Info</button>
<button type="button" class="btn btn-dark">Dark</button>
<button type="button" class="btn btn-light">Light</button>
<button type="button" class="btn btn-default">Default</button>
<button type="button" class="btn btn-link">Link</button>

Outline buttons

In need of a button, but not the hefty background colors they bring? Replace the default modifier classes with the .btn-outline-* ones to remove all background images and colors on any button.

<button type="button" class="btn btn-outline-primary">Primary</button>
<button type="button" class="btn btn-outline-secondary">Secondary</button>
<button type="button" class="btn btn-outline-success">Success</button>
<button type="button" class="btn btn-outline-danger">Danger</button>
<button type="button" class="btn btn-outline-warning">Warning</button>
<button type="button" class="btn btn-outline-info">Info</button>
<button type="button" class="btn btn-outline-dark">Dark</button>
<button type="button" class="btn btn-outline-light">Light</button>
<button type="button" class="btn btn-outline">Outline</button>

Button tags

The .btn classes are designed to be used with the <button> element. However, you can also use these classes on <a> or <input> elements.

Link 
<a class="btn btn-primary" href="#" role="button">Link</a>
<button class="btn btn-primary" type="submit">Button</button>
<input class="btn btn-primary" type="button" value="Input">
<input class="btn btn-primary" type="submit" value="Submit">
<input class="btn btn-primary" type="reset" value="Reset">

Sizes

Add the .btn-sm or .btn-lg class to a button to make it smaller or larger just like in the example below.

<button type="button" class="btn btn-outline btn-lg">Large button</button>
<button type="button" class="btn btn-outline">Default button</button>
<button type="button" class="btn btn-outline btn-sm">Small button</button>
<button type="button" class="btn btn-outline btn-xs">Extra small button</button>

Rounded

It is also possible to use rounded buttons by adding .btn-rounded class to the button. It can be useful when we would like to display the button in different style.

<button type="button" class="btn btn-outline btn-lg btn-rounded">Large button</button>
<button type="button" class="btn btn-outline btn-rounded">Default button</button>
<button type="button" class="btn btn-outline btn-sm btn-rounded">Small button</button>
<button type="button" class="btn btn-outline btn-xs btn-rounded">Extra small button</button>

Icons

Easy to use buttons with icons just add the requested icon with aligned class. Add .mr-* class to the icon if you would like to use a small margin.

<button type="button" class="btn btn-primary">Primary <i class="ya ya-shopping-cart"></i></button>
<button type="button" class="btn btn-success btn-icon-left"><i class="ya ya-check-circle"></i> Success</button>
<button type="button" class="btn btn-danger">Danger <i class="ya ya-bell"></i></button>
<button type="button" class="btn btn-warning">Warning <i class="ya ya-email"></i></button>
<button type="button" class="btn btn-info btn-icon-left"><i class="ya ya-gear"></i> Info</button>
<button type="button" class="btn btn-dark btn-icon-left"><i class="ya ya-like"></i> Dark</button>
<button type="button" class="btn btn-light btn-icon-left"><i class="ya ya-comments"></i> Light</button>
<button type="button" class="btn btn-link btn-icon-left"><i class="ya ya-attachment"></i> Link</button>

Icon buttons

Use .btn-icon class for buttons when you would like to display only icon in the button. This will add a smaller padding to the button.

<button type="button" class="btn btn-primary btn-icon"><i class="ya ya-shopping-cart"></i></button>
<button type="button" class="btn btn-secondary btn-icon"><i class="ya ya-comments"></i></button>
<button type="button" class="btn btn-success btn-icon"><i class="ya ya-check-circle"></i></button>
<button type="button" class="btn btn-danger btn-icon"><i class="ya ya-bell"></i></button>

Button with counts

We can easily append a count to any kind of button. Use the selected button in a .btn-group and add the selected size with number just like in the example below.

Watch 45
Like 138 
<div class="btn-group mr-1">
    <a class="btn btn-default btn-xs btn-count" href="#" role="button"><i class="ya ya-eye"></i> Watch</a>
    <a class="btn btn-light btn-xs" href="#" role="button">45</a>
</div>

<div class="btn-group">
    <a class="btn btn-default btn-xs btn-count" href="#" role="button"><i class="ya ya-heart-add"></i> Like</a>
    <a class="btn btn-light btn-xs" href="#" role="button">138</a>
</div>

Button groups

Have a hankering for a series of buttons that are attached to one another? Wrap them in a .btn-group and the buttons will be rounded and spaced automatically.

<div class="btn-group">
    <a class="btn btn-default btn-icon" href=""><i class="ya ya-small-left"></i></a>
    <a class="btn btn-default btn-icon" href=""><i class="ya ya-small-right"></i></a>
</div>

<div class="btn-group" role="group">
    <button type="button" class="btn btn-light"><i class="ya ya-list m-0"></i></button>
    <button type="button" class="btn btn-light"><i class="ya ya-table m-0"></i></button>
</div>

Comments

    <a class="btn btn-outline btn-icon" href="#" role="button"><i class="ya ya-like"></i></a>

</div>

<div class="btn-group" role="group">
    <a class="btn btn-outline" href="#" role="button">Comments</a>
    <a class="btn btn-outline btn-icon" href="#" role="button"><i class="ya ya-comments"></i></a>

States

Disable <button> elements with the boolean disabled attribute and <a> elements with the .disabled class.

Active button 
<button type="button" class="btn btn-primary" disabled>Disabled button</button>
<a href="#" class="btn btn-primary active" role="button" aria-pressed="true">Active button</a>

Block buttons

Make any button full-width by adding .btn-block. It adds width: 100%;, changes the display from inline-block to block, and centers the button text.

Block button Small block button 
<a class="btn btn-outline btn-block" href="#" role="button">Block button</a>
<a class="btn btn-outline btn-block btn-sm" href="#" role="button">Small block button</a>
