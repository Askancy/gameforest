Introduction

Gameforest's built with Bootstrap 4 ─ the world’s most popular framework for building responsive, mobile-first sites and applications. Inside you’ll find high quality HTML, CSS, SASS, TWIG and JavaScript files to make your own gaming website easier than ever.

Quick start

Gameforest is a static html template but the best way to build your own website and customize the theme if you are using the built in theme features. For more information choose from the sidebar sections.

HTML markup

The basic template is a guideline for how to structure your pages when building with the Gameforest theme. Included below are all the necessary bits for using the theme’s CSS and JS.

<!DOCTYPE html>
<html lang="en" dir="ltr">
<head>
    <!-- meta -->
    <meta charset="utf-8">
    <meta name="viewport" content="width=device-width, initial-scale=1, shrink-to-fit=no">

    <!-- Title -->
    <title>Gameforest Bootstrap Gaming Theme</title>

    <!-- css: theme -->
    <link rel="stylesheet" href="css/theme.min.css">
</head>
<body class="fixed-navbar">
    <div class="site-content">...</div>

    <!-- js: vendor -->
    <script src="plugins/jquery/jquery.min.js"></script>
    <script src="plugins/bootstrap/js/bootstrap.bundle.min.js"></script>

    <!-- js: theme -->
    <script src="js/theme.bundle.min.js"></script>
</body>
</html>

Twig markup

We can use twig.js to build our html pages. This is the recommended way because twig has several advantages what was used in the build process.

{% extends "base.twig" %}

{% block body %}
    <section>
        <div class="container">
            <div class="row">
                <div class="col">
                    ...
                </div>
            </div>
        </div>
    </section>
{% endblock %}
