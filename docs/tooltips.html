Tooltips

Documentation and examples for adding custom tooltips with CSS and JavaScript using CSS3 for animations and data-attributes for local title storage.

Example

The tooltip plugin generates content and markup on demand, and by default places tooltips after their trigger element.

<button type="button" class="btn btn-outline" data-toggle="tooltip" data-placement="top" title="Tooltip on top">
    Tooltip on top
</button>

<button type="button" class="btn btn-outline" data-toggle="tooltip" data-placement="right" title="Tooltip on right">
    Tooltip on right
</button>

<button type="button" class="btn btn-outline" data-toggle="tooltip" data-placement="bottom" title="Tooltip on bottom">
    Tooltip on bottom
</button>

<button type="button" class="btn btn-outline" data-toggle="tooltip" data-placement="left" title="Tooltip on left">
    Tooltip on left
</button>

Forms

It is also possible to add a tooltip to our forms and inputs. In this example we added our tooltip to an input group but you can add it anywhere.

<div class="input-group">
    <input type="text" class="form-control" placeholder="Enter title" aria-label="Example text with icon addon" aria-describedby="button-addon1">
    <div class="input-group-append">
        <span class="input-group-text bg-transparent border-0" id="inputGroup-sizing-sm" data-toggle="tooltip" data-placement="top" title="Tooltip on top"><i class="ya ya-help-circle"></i></span>
    </div>
</div>
