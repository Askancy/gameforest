Disqus

Add disqus plugin as a comment section to the page by using ya-disqus attribute with the right url just like in the example below.

<div class="comments">
    <div ya-disqus></div>
</div>

Facebook

Add facebook comment plugin to the page by using ya-facebook attribute with the right url just like in the example below.

<div class="comments">
    <div ya-facebook></div>
</div>
