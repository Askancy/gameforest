Avatars

Avatars are images that users can set as their profile picture. They can be custom photos, uploaded by users, or generated as Identicons as a placeholder.

Example

Add .avatar to any img element to make it an avatar. For different sizes you can add .avatar-sm or .avatar-md class to the avatar.

<img class="avatar avatar-xl" src="img/user.jpg" alt="">
<img class="avatar avatar-lg" src="img/user.jpg" alt="">
<img class="avatar" src="img/user-1.jpg" alt="">
<img class="avatar avatar-md" src="img/user-1.jpg" alt="">
<img class="avatar avatar-sm" src="img/user-1.jpg" alt="">
<img class="avatar avatar-xs" src="img/user-1.jpg" alt="">

Tile

We can display avatars with custom text like user name or title by using .avatar-tile in a href element. In this case you don't have to add avatar class to the image element.

Nathan Drake @nathan 
<a class="avatar-tile" href="#">
    <img src="img/user-1.jpg" alt="">
    <div>
        <strong>Nathan Drake</strong>
        <span class="d-block">@nathan</span>
    </div>
</a>

Card

Display avatar as a small card by adding .avatar-card component to the page. We can also add custom text and title to this component.

Smallville

gameforest author

 
<a class="avatar-card" href="#">
    <div>
        <div class="avatar-title">Smallville</div>
        <p class="avatar-text">gameforest author</p>
    </div>
    <imgsrc="img/user-5.jpg" alt="">
</a>

Thumbnail

Use .avatar-thumbnail class with img element to add a small padding and border with white background to the image. It can be useful on the profile page.

<img class="avatar-thumbnail avatar-xl" src="img/user.jpg" alt="">
<img class="avatar-thumbnail avatar-lg" src="img/user.jpg" alt="">
<img class="avatar-thumbnail" src="img/user-1.jpg" alt="">
<img class="avatar-thumbnail avatar-md" src="img/user-1.jpg" alt="">
<img class="avatar-thumbnail avatar-sm" src="img/user-1.jpg" alt="">
<img class="avatar-thumbnail avatar-xs" src="img/user-1.jpg" alt="">

Badge

Use .avatar-badge class with div element to add a custom badge or status to the image. It can be useful on any listing page.

<div class="avatar-badge">
    <img src="img/user-5.jpg" alt="">
    <a class="badge badge-primary" href="#" data-toggle="tooltip" data-placement="bottom" title="Add friend"><i class="ya ya-user-add"></i></a>
</div>

<div class="avatar-badge">
    <img src="img/user-5.jpg" alt="">
    <span class="badge badge-success"></span>
</div>

Stack

Stacked avatars can be used to show multiple collaborators or participants when there is limited space available.

<div class="avatar-stack">
    <img class="avatar" src="img/user-1.jpg" alt="">
    <img class="avatar" src="img/user-2.jpg" alt="">
    <img class="avatar" src="img/user-3.jpg" alt="">
    <img class="avatar" src="img/user-4.jpg" alt="">
    <img class="avatar" src="img/user-5.jpg" alt="">
</div>

Stack with text

It is also possible to use custom text with avatar-stack by adding .avatar-text to the avatar element just like in the example below.

Elizabeth,Venom and more 45 people like it. 
<div class="avatar-stack">
    <img class="avatar" src="img/user-1.jpg" alt="">
    <img class="avatar" src="img/user-2.jpg" alt="">
    <img class="avatar" src="img/user-3.jpg" alt="">
    <img class="avatar" src="img/user-4.jpg" alt="">
    <img class="avatar" src="img/user-5.jpg" alt="">
    <span class="avatar-text">
        <a href="profile.html">Elizabeth</a>,<a href="profile.html">Venom</a> and more 45 people like it.
    </span>
</div>
