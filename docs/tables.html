Tables

Documentation and examples for opt-in styling of tables. Display custom data like users, posts etc in tables by choosing from several styles.

Examples

To apply this component, add the .table class to a <table> element.

Trevor trevor@gtav.com admin
Elizabeth elizabeth@gs.com member
