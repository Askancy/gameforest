Popovers

Documentation and examples for adding Bootstrap popovers, like those found in iOS, to any element on your site.

Example

Add popover to the button by using data-toggle="popover" attribute and with the necesary attributes like in the example below.

<button type="button" class="btn btn-outline" data-toggle="popover" title="Popover title" data-content="And here's some amazing content. It\'s very engaging. Right?">Click to toggle popover</button>

Positions

Use data-placement attribute with the chosen position on the element to set the right position for your popover.

<button type="button" class="btn btn-outline" data-container="body" data-toggle="popover" data-placement="top" data-content="Vivamus sagittis lacus vel augue laoreet rutrum faucibus.">
  Popover on top
</button>

</button>
