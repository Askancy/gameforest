File Structure

Within the download you'll find the following directories and files, logically grouping common assets and providing both compiled and minified variations.

Root

We can find a dist, src and docs folder with the necessary packages, linters and config files in the root for our project but it is not recommended to touch core files because of the upcoming updates in the future.

gameforest/
└── dist/
└── docs/
└── src/
└── *
  • ./dist contains all of files what was compiled from src folder and it is required to use for our project. We can upload files from dist folder to our webspace via ftp client.
  • ./src folder includes the source code and files of our theme. We should never share the included source files with 3rd person on our webspace.
  • ./docs folder includes the documentation of theme with important instructions to getting started with your project.

HTML

In the dist folder we can find the required html files for our project. These files are built by twig.js, a modern javascript template engine.

gameforest/
└── dist/
    └── *.html

Included more than 50 html files in this directory and easy to modify them. For more information please check the twig section.

Twig

Gameforest is built by Twig what is an easy way to create html files and modify theme with a single task. Twig files are available in ./src/twig folder.

gameforest/
└── src/
    ├── json
    └── twig
        ├── views
        └── *.twig
  • ./src/json contains all of files with data and examples what was required to build our theme. Just an example for our blogs, the text and data like comments or dates are coming from blogs.json
  • ./src/twig contains all of twig files what is important to compile single html files into dist folder. It is also possible to watch these files with npm tasks.
  • ./src/twig/views folder includes the basic twig files for html attributes like header and footer.

SCSS

Scss contains the source for all bootstrap and theme css. Utilize our source Sass files to take advantage of variables, maps, mixins, and more.

gameforest/
└── src/
    └── scss
        └── child
            └── _theme.scss
            └── _variables.scss
        └── parent
            └── mixins
            └── plugins
            └── utilites
            └── *.scss
        └── theme.scss
  • ./src/scss contains all of scss files what is required to build our theme css file. Compiled and minifed css files are available in ./dist/css folder.
  • ./src/scss/child contains all of custom scss files for customization and theme modifications.
    • ./src/scss/child/_theme.scss import custom scss files and add extra classes to the theme in this file.
    • ./src/scss/child/_variables.scss overwrite the theme variables by adding custom theme variables without !default tag.
  • ./src/scss/mixins contains all of mixins for our scss files.
  • ./src/scss/plugins folder includes the scss files and theme customization for 3rd party plugins.

Javascript

Javascript files can be found in the dist and src folder. In ./dist/js folder we've compiled and minified versions of es6 js files.

gameforest/
└── src/
    └── js
        └── *.js

Javascript files are built with es6 and for more information check the javascript section.

  • theme.js contains all of javascript files from src folder what was translated with rollup plugin.
  • theme.min.js is the minified version of theme.js.
  • theme.bundle.js contains all of the custom plugins and additional scripts included in.
  • theme.bundle.min.js is the minified version of theme.bundle.js that should be included in your website.
