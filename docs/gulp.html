Gulp

Gulp is a toolkit for automating painful or time-consuming tasks in your development workflow, so you can stop messing around and build something.

How to install

In order to use Gulp you will need to download Node JS. If node is not installed on your computer then click here to download. Once node is installed, run the following npm command to install gulp client globally.

npm install gulp -g

Dependencies

After gulp was installed globally you need to run the following command to install the dev dependencies. Just like node.js, yarn is also required to be installed on your computer for usage.

yarn

It will install the dev and regular dependencies to our theme directory.

Start to build

We are ready to run a development environment by typing yarn start command into the terminal. Gameforest will be opened in a new browser tab.

yarn start

start task is watching ./src folder to see current file changes and your browser will be reloaded if you've made a successful modification.

Available Tasks

It is also possible to run single gulp task from terminal by typing yarn * when made a small modification just choose from tasks below.

Name Description
start Running a development environment for our project.
build Run css and js task together.
css Compile scss files into minified and unminified css.
js Build javascript files with babel.
icon Generate icon fonts from svg.
html Render twig files into html output.
image Optimize images from src to dist folder.
plugins Move available plugins from node to dist.
docs Run the docs folder on localhost.
