Input Groups

Easily extend form controls by adding text, buttons, or button groups on either side of textual inputs, custom selects, and custom file inputs.

Example

Place one add-on or button on either side of an input. You may also place one on both sides of an input.

.yakuthemes.com
https://yakuthemes.com/ 
<div class="input-group mb-3">
    <div class="input-group-prepend">
        <span class="input-group-text" id="basic-addon1"><i class="ya ya-user"></i></span>
    </div>
    <input type="text" class="form-control" placeholder="Username" aria-label="Username">
</div>

<div class="input-group mb-3">
    <input type="text" class="form-control" placeholder="Website" aria-label="Website">
    <div class="input-group-append">
        <span class="input-group-text" id="basic-addon2">.yakuthemes.com</span>
    </div>
</div>

<div class="input-group mb-3">
    <div class="input-group-prepend">
        <span class="input-group-text" id="basic-addon1">https://yakuthemes.com/</span>
    </div>
    <input type="text" class="form-control" id="basic-url" placeholder="username">
</div>

Sizing

Add the relative form sizing classes to the .input-group itself and contents within will automatically resize—no need for repeating the form control size classes on each element.

Large
Default
Small 
<div class="row">
    <div class="input-group input-group-lg col-lg-4">
        <div class="input-group-prepend">
            <span class="input-group-text" id="inputGroup-sizing-lg">Large</span>
        </div>
        <input type="text" class="form-control" aria-label="Sizing example input">
    </div>

    <div class="input-group mb-3 col-lg-4">
        <div class="input-group-prepend">
            <span class="input-group-text" id="inputGroup-sizing-default">Default</span>
        </div>
        <input type="text" class="form-control" aria-label="Sizing example input">
    </div>

    <div class="input-group input-group-sm mb-3 col-lg-4">
        <div class="input-group-prepend">
            <span class="input-group-text" id="inputGroup-sizing-sm">Small</span>
        </div>
        <input type="text" class="form-control" aria-label="Sizing example input">
    </div>
</div>

Multiple addons

Multiple add-ons are supported and can be mixed with checkbox and radio input versions.

$ 0.00 
<div class="input-group">
    <div class="input-group-prepend">
        <span class="input-group-text">$</span>
        <span class="input-group-text">0.00</span>
    </div>
    <input type="text" class="form-control" aria-label="Dollar amount (with dot and two decimal places)">
</div>

Button addons

Add button to the input group and use it just like in the example below.

<div class="input-group">
    <div class="input-group-prepend">
        <button class="btn btn-outline" type="button" id="button-addon1">Button</button>
    </div>
    <input type="text" class="form-control" placeholder="" aria-label="Example text with button addon">
</div>

Icon addons

Add selected background with .bg-* class to the .input-group-text and remove border from .form-control to get this results like below.

<div class="row">
    <div class="input-group col">
        <div class="input-group-prepend">
            <button type="button" class="btn btn-light btn-icon border-right-0" data-toggle><i class="ya ya-user"></i></button>
        </div>
        <input type="text" class="form-control pl-2" placeholder="Enter username" aria-label="Example text with icon addon">
    </div>

    <div class="input-group col">
        <input type="password" class="form-control" placeholder="Enter password" aria-label="Example text with icon addon">
        <div class="input-group-append">
            <button type="button" class="btn btn-light btn-icon border-left-0" data-toggle><i class="ya ya-lock"></i></button>
        </div>
    </div>
</div>
