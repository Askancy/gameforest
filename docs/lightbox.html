Lightbox

Add image to the current page in a popup box by using gameforest lightweight lightbox component. Easy to use and can be customized in several ways.

Example

To apply the lightbox component just add the ya-lightbox attribute to an element to turn all anchors from element into lightbox links.

Open lightbox 
<a class="btn btn-outline" href="https://static.giantbomb.com/uploads/original/5/56742/3058974-3631271585-rdr2-.jpg" ya-lightbox role="button">Open lightbox</a>

Gallery

Add images into a gallery with the lightbox component by using the rel attribute. It will append images to a gallery where rel attribute values are equals.

Video

To add a video to the lightbox, use video url in a href attribute. It is also possible to use customizations from embed component.

Comments

We can display user comments with lightbox plugin. Recommended to add a custom url to the opened image just like in the example below.

Options

option value default description
disqus string null Add disqus comment plugin to the lightbox by adding disqus url.
facebook string null Add facebook comment plugin to the lightbox by adding facebook api id.
