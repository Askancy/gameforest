Icons

Documentation and examples for icons, nucleo premium icon library. Add the chosen icon to custom components like button or cards etc.

Premium Icons

Included hundred icons from nucleo premium icon library. You can reach these icons with .ya .ya-* classes inside in i html attribute. If you can't find your needed icon then you are still able to choose from fontawesome icons.

<i class="ya ya-*"></i>

Sizes

It is also possible to display icons in different sizes. Add .ya-md or .ya-lg and .ya-xl class to the icon and the icon will be visible in choosen size.

<i class="ya ya-helmet"></i>
<i class="ya ya-helmet ya-md"></i>
<i class="ya ya-helmet ya-lg"></i>
<i class="ya ya-helmet ya-xl"></i>

SVG Icon

Svg can be useful when we would like to use more than one color for our icon. Use ya-svg attribute with selected svg icon from the example below.

<span ya-svg="awesome"></span>
<span ya-svg="like"></span>
<span ya-svg="sick"></span>
<span ya-svg="angry"></span>

FontAwesome 5

Fontawesome stylesheet will be added automatically to the header when we use an icon. Check the icon library for more examples.
